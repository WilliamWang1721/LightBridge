# LightBridge 开发 AGENT 文档

## 开发参考
每当你开发一个新功能时，请浏览当前文件夹下的“参考项目”，该文件夹中具有多个成熟开源项目的代码供你使用，若没有强制要求，能够直接从中借鉴的代码请直接使用，无需再次重做。使用的代码请在README.md文档中注明。

请阅读项目整体开发文档以确保你的开发处于正规。开发文档位于当前项目的参考资料文件夹中。

## 开发记录
请你在完成一个任务之后，在当前项目文件夹中的“参考资料”文件夹内，在“项目日志”文档中按照规定格式写入必要信息并做归档。

## 对话要求
每次与我对话时，请务必使用简体中文。且每次对话前，请在开头加入"当前对话已使用 agent.md:"
